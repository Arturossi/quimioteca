Sobre

A plataforma da Quimioteca Nacional conta com um website que mostra ao usuário informações relevantes sobre todas as moléculas armazenadas em seu banco de dados.

Implementação

Desenvolvida com os Frameworks Django foi criada uma solução completa escrita em Python que utiliza o padrão de desenvolvimento model-template-view (MTV). Para a parte visual do sistema foi utilizado o Pigendo que fornece componentes de interface front-end para sistemas utilizando HTML, CSS e JAVASCRIPT melhorando a comunicação com o usuário fornecendo uma plataforma limpa e fácil de se usar.

Origens

A Quimioteca Nacional foi concebida pelo pesquisador Diego Gomes dureante pós-doutorado no Institut de Science et d'Ingénierie Supramoléculaires da Université de Strasbourg incentivada pelos professores Jean Marie Lehn e Jean-Pierre Sauvage. O plano diretor foi concebido sob orientação do prof. Marcel Hibert da Faculte de Pharmacie fundador da Chimiotequè Nationale du CNRS
